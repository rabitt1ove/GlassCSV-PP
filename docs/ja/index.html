データをそのまま保持

型変換による0落ちや日付の自動変更を防止。「001」を「1」に変えることなく、データのありのままを「文字」として忠実に保持します。

Macネイティブの速度

Swift6とAppKitを駆使したネイティブ設計。巨大なファイルも待たせることなく、常に軽快なレスポンスで表示・検索が可能です。
